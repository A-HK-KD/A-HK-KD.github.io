System prompt · ~800 tokens
"You are a performance assessment assistant for PAT. Generate STAR-format draft responses…"
Loaded from s3://pat-ai-prompts/self-assessment/v1-system.txt. Sets role, output format, tone, and STAR structure rules. Versioned — any change requires a new template version key to preserve cache stability. Never modified mid-cycle.
Output schema · ~400 tokens
JSON schema: { question_id, star_draft: { situation, task, action, result } }
Strict JSON schema injected as a second system block. Nova Lite follows JSON-only output reliably when the schema is explicit. Includes field descriptions and length constraints (situation ≤ 80 words, result must include a metric).
cache checkpoint 1 — 1,200 tokens · threshold met (min 1,024) · TTL 5 min
Retrieved — dev plan chunks · ~1,800 tokens
Requirement: "Deliver microservices migration" · Status: complete · Progress: "…"
Top-10 chunks returned from hybrid search (semantic + BM25) against the employee's dev plan. Cross-encoder re-ranked. Chunked at requirement level, ~100–300 tokens each. Metadata includes completion timestamp, tags, and status. Static within a single pre-generation batch.
Retrieved — past assessments · ~800 tokens
H2 25 Q3: "I led the backend refactor by coordinating with three teams…"
2–3 past self-assessment responses retrieved for stylistic grounding. Semantic chunks at sentence/paragraph boundaries, 300–500 tokens each with 50-token overlap. Provides writing style context and grounds the draft in the employee's own voice. Used by "Match My Persona" in phase 2.
cache checkpoint 2 — 3,800 tokens · all content above here reused across Q1–10 · TTL 5 min
Style profile · ~250 tokens · (phase 2)
avg_sentence_len: 18 · tone: formal · person: first · vocab_complexity: 0.72
Extracted offline by a separate Lambda that analyses 2–3 past self-assessments. Stored in ai-style-profiles-{stage} DynamoDB table, persistent across cycles. Injected at runtime just before the user turn. Increases context size; scoped to phase 2.
User turn — question · ~20 tokens
"What were your key projects for this half, and how did you contribute to each one?"
The only part of the prompt that changes between the 10 questions in a pre-generation batch. ~15–25 tokens. Because it sits after checkpoint 2, every token above it in the context is served from cache — the model only processes this tiny delta per question.
